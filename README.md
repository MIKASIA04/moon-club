##🌙 Moon Club

![Moon Club Homepage] (images/moon-club-preview.png)

My first web development project, yay!

Moon Club is a moon-themed website I built while learning HTML and CSS. Instead of creating another generic practice project, I wanted to build something around a topic I genuinely enjoy—the Moon.

The website includes a landing banner, responsive navigation, moon imagery, and a piece of writing dedicated to the Moon and what it represents to me. The project was mainly an opportunity to explore web development fundamentals while creating something a little more personal.

## What I used :

-HTML5
-CSS3
-Flexbox
-Media Queries
-Responsive Design



## What I Learned:

While building Moon Club, I spent a lot of time experimenting, breaking things, fixing them, and figuring out why they broke in the first place. Through that process, I got hands-on experience with:

-Semantic HTML
-Fixed and sticky positioning
-Flexbox layouts
-Responsive design principles
-Media queries
-Hover effects and transitions
-Form validation
-The CSS box model

One of the biggest lessons from this project was understanding responsiveness. Making a website look good on a large screen is one thing; making it work properly across tablets and phones is a completely different challenge.

## Features

- Responsive layout across multiple screen sizes
- Sticky navigation bar
- Interactive image hover effects
- Email input with validation
- Custom moon-themed content
- Mobile-friendly design

## Why I Built It

Honestly, because I like the Moon.

Most beginner projects are calculators, landing pages, or clones of existing websites. I wanted to build something that felt a little more personal and reflected an interest of mine while still helping me strengthen my front-end skills.

Moon Club may be a simple project, but it taught me a lot about how websites are structured, styled, and adapted for different devices.
